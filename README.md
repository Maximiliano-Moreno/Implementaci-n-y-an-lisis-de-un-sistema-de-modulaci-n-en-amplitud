La modulación en amplitud (AM) es una de las técnicas de modulación más fundamentales en el campo de las comunicaciones analógicas, utilizada ampliamente en transmisiones de radio comercial, comunicaciones aeronáuticas y sistemas de radiodifusión. Este proyecto implementa un sistema completo de modulación en amplitud utilizando Python, con el objetivo de analizar el comportamiento de la señal modulada bajo diferentes condiciones de transmisión, incluyendo ruido, atenuación y distorsión por sobremodulación.

El desarrollo de este sistema permite comprender los principios fundamentales de la modulación AM, donde una señal de baja frecuencia (mensaje) modula la amplitud de una señal portadora de alta frecuencia, permitiendo su transmisión eficiente a través de canales de comunicación. El análisis se realiza tanto en el dominio del tiempo como en el dominio de la frecuencia, utilizando herramientas como NumPy para el procesamiento numérico, SciPy para el diseño de filtros y Matplotlib para la visualización gráfica de resultados.

A lo largo de este proyecto, se exploran conceptos clave como el índice de modulación, el ancho de banda de la señal AM, el efecto del ruido aditivo blanco gaussiano (AWGN), la atenuación en el canal de transmisión y la distorsión introducida por sobremodulación. Además, se implementa un detector de envolvente para la demodulación de la señal, evaluando la calidad de recuperación mediante el error cuadrático medio (MSE).

El código desarrollado es completamente interactivo, permitiendo ajustar parámetros como la frecuencia de la señal de mensaje, la frecuencia de la portadora, el índice de modulación, la relación señal-ruido (SNR) y el nivel de atenuación, facilitando así un análisis exhaustivo del comportamiento del sistema en diversos escenarios de comunicación reales.

Este proyecto representa una herramienta educativa valiosa para estudiantes y profesionales interesados en comprender los fundamentos de los sistemas de comunicación analógicos, así como para aquellos que buscan implementar simulaciones de sistemas de modulación en Python. El código está completamente documentado y organizado en celdas para su fácil ejecución en entornos como Google Colab o Jupyter Notebook.
